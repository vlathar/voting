USER MANUAL FOR LNMIIT ONLINE VOTING SYSTEM




The web application LNMIIT ONLINE VOTING SYSTEM has been developed for the LNMIIT, Jaipur to reduce the manual hefty process of Elections to a light and user interactive session.

Following are the instructions to avail various functionalities offered by the application as per three categories of users:

1. Member of Student Gymkhana

Login using a valid username and password to perform the following functions:

1.1. Cast Vote

1.2 Apply for Candidature

(i) Scroll down and click on “Apply for candidature”.
(ii) Fill the form displayed.
(iii)Click “Apply”.

2. Contesting Candidates

Login using a valid username and password to perform the following functions:

2.1 Manage portfolio

(i) Scroll down and click on “Update portfolio”.

(ii) Fill the incomplete fields and edit the filled ones if so required.

(iii) Click “Update”.

2.2 Cancel candidature

(i) Scroll down and click on “DELETE YOUR CANDIDATURE”.

(ii) Press “Delete”.

3. The Chief Election Officer (CEO)

Login using the username and password for CEO to avail following functions .

3.1 Manage Election Event

3.1.1 Create Election Event

(i) Scroll down and click on “CREATE ELECTION EVENT”.

(ii) In the form displayed, fill the name, date, start time and end time fields.

(iii) Choose the positions to hold elections for.

(iv) Click “CREATE”.

3.1.2 Update Election Event

(i) Scroll down and click on “UPDATE ELECTION EVENT”.

(ii) Choose the event to be updated.

(iii) Edit fields if required.

(iv) Click “UPDATE”.

3.1.3 Delete Election Event

(i) Scroll down and click on “DELETE ELECTION EVENT”.

(ii) Choose the event to be deleted in the form displayed.

(iii) Click “DELETE”.

3.2. Manage Candidature Applications

(i) Scroll down and click on “VIEW CANDIDATURE APPLICATIONS”.

(ii) From the selectable list of roll numbers displayed, choose the roll number of the candidate whose application has to be viewed.

(iii)Application of the respective candidate is displayed.

  1. Click “APPROVE” to approve an application for candidature.

  2. Click “DISAPPROVE” to disapprove an application for candidature.

3.3 Publish Results

(i) Click on “PUBLISH RESULTS” to announce the election results.
